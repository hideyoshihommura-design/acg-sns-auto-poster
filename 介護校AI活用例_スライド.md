## スライド15：活用例5 NotebookLMとは

**タイトル**
```
NotebookLMとは？
```

**本文**
```
Googleが提供する無料AIツール

📄 マニュアルをアップロードするだけで
   「その資料だけを参照して回答するAI」が作れる

🔒 資料以外の情報は使わない → 嘘の回答を防げる

🎧 Audio Overview：資料をポッドキャスト風の音声に自動変換
📊 スライド自動生成：研修用プレゼンをPPTX形式で出力
💬 質問応答：スタッフがいつでも質問できる
```

## スライドデザインのガイドライン

### カラーパレット
| 用途 | カラー | 使い方 |
|------|--------|-------|
| メインカラー | #1e3a5f（ダークブルー） | タイトル、見出し |
| アクセント | #dc2626（レッド） | CTA、強調 |
| ポジティブ | #16a34a（グリーン） | After、効果 |
| ネガティブ | #6b7280（グレー） | Before |
| 背景 | #ffffff / #f8fafc | 本文スライド / セクション区切り |

### フォントサイズ目安
| 要素 | サイズ |
|------|-------|
| スライドタイトル | 36〜44pt |
| サブタイトル | 20〜24pt |
| 本文 | 18〜20pt |
| 補足・出典 | 14pt |

### Before/Afterスライドの共通ルール
- 左右2カラムで配置
- Before: グレー背景（#f3f4f6）に ❌ アイコン
- After: 薄いグリーン背景（#f0fdf4）に ✅ アイコン
- 下部に「📊 効果ハイライト」を1〜2行で配置
